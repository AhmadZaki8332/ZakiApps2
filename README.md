# 🍔 Food Delivery App (UI Prototype)

Sebuah purwarupa (*prototype*) aplikasi pemesanan makanan berbasis Android. Proyek ini difokuskan pada pengembangan **Front-End** dan eksplorasi antarmuka pengguna (UI/UX) yang modern, bersih, dan responsif. 

*Catatan: Aplikasi ini saat ini hanya mencakup tampilan sisi klien (Front-End) dan menggunakan data statis (dummy data). Belum terhubung dengan REST API atau database Backend.*

## 🛠️ Teknologi yang Digunakan
*   **Bahasa Pemrograman:** Kotlin
*   **UI Toolkit:** Jetpack Compose

## ✨ Fitur Tampilan
*   **Beranda:** Menampilkan daftar menu favorit dan restoran terdekat.
*   **Menu & Filter:** Halaman eksplorasi makanan dengan opsi filter pencarian.
*   **Detail Makanan:** Menampilkan gambar makanan, deskripsi, lokasi, dan tombol pemesanan.
*   **Keranjang/Pesanan:** Halaman ringkasan pesanan pengguna.
